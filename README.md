# Beta Granite Website

A modern, responsive website showcasing premium granite products and services.

## Project Structure

```
beta-granite-website/
├── index.html      <- Main homepage file
├── style.css       <- CSS styling for the website
├── images/         <- Directory for hero and slab images
│   ├── hero.jpg
│   ├── slab1.jpg
│   └── slab2.jpg
└── README.md       <- Project documentation
```

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Clean and professional granite-themed design
- **Product Gallery**: Showcase granite slabs and products
- **Contact Section**: Easy way for customers to get in touch
- **Hero Section**: Eye-catching banner with granite imagery

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with flexbox and responsive design
- **Responsive Images**: Images in the `images/` folder for optimal performance

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/endritbeta/beta-granite-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd beta-granite-website
   ```

3. Open `index.html` in your web browser to view the website

4. Add your granite images to the `images/` folder

## File Descriptions

- **index.html**: Main HTML file containing the structure and content
- **style.css**: All CSS styling for layout, colors, and responsiveness
- **images/**: Directory to store hero images and granite slab photos

## Customization

You can easily customize:
- Colors and fonts in `style.css`
- Content and sections in `index.html`
- Add your own images to the `images/` folder
- Update contact information
